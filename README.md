# ZET - Digital Solutions

A modern React web application showcasing digital solutions and services.

## Project Structure

```
├── public/
│   └── index.html          # HTML entry point
├── src/
│   ├── App.jsx            # Main React component
│   ├── App.css            # App styling
│   └── index.js           # React DOM entry point
├── package.json           # Dependencies and scripts
└── vercel.json           # Vercel deployment config
```

## Getting Started

### Prerequisites
- Node.js 14+ and npm

### Installation
```bash
npm install
```

### Development
```bash
npm start
```
The app will open at `http://localhost:3000`

### Build
```bash
npm run build
```
Creates an optimized production build in the `build/` folder.

## Deployment

### Vercel Deployment
1. Push your code to GitHub
2. Go to [Vercel](https://vercel.com)
3. Click "New Project"
4. Select this repository
5. Vercel will auto-detect your React app and deploy it

The `vercel.json` configuration handles the build and output settings automatically.
